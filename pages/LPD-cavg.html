git fetch

Sincroniza,el repositorio local y el repositorio remoto, ademas sirve para poder descargar: commits, archivos, referencias e incluso ramas. Para poder hacer las descargar se necesita del enlace del repositorio remoto, si solo se quiere descargar un componente especifico del repositorio se deberia usar estenciones como el --all, bracnh, remote ect.
