Git reset

Git reset=Elimina las dades del intercambi temporal, però hi han tres variacions més diferents:

Git reset fichero=elimina els canvis del fitxer de la zona d'intercanvi temporal, però no cambia res en el directori de treball Git reset --hard commit=elimina tots els canvis des del commit commit i actualitza el HEAD aquest commit. Git reset commit=elimina els posterios comit,però no elimina els canvis del directori de treball.
