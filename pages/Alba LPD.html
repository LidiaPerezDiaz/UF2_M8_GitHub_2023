UF2 - Comandes Git

Git reset

Elimina els canvis de la zona d'intercanvi temporal. També hi han quatre variacions que són: git reset "fichero"(elimina els canvis del fitxer de la zona d'intercanvi temporal, però preserva els canvis en el directori de treball), git reset --hard "commit"(elimina tots els canvis des del commit i actualitza el HEAD aquest commit), git reset "commit"(elimina tots els commits posteriors a aquest commit).
